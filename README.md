
# Overview

Repo: [https://github.com/slingr-stack/scanner-package](https://github.com/slingr-stack/scanner-package)

This package allows your SLINGR application to scan QR and barcodes.

## QuickStart

```js
pkg.scanner.functions.openScanner();
```
